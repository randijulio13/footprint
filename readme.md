# Footprint

## Tech Stacks
This app is built using [ReactJS](https://react.dev/) with [Vite](https://vitejs.dev/) as bundler and using [MaterialUI](https://mui.com/) Framework.

## Feature
```
1. Create New Post
2. Delete Post
3. Like Post
4. Unlike Post
5. Comment Post
6. Delete Comment
7. Follow User
8. Unfollow User
9. Send Chat
10. Dark Mode
```
